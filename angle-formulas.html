معادلات حساب الزوايا

الأسس الرياضية لضبط الأطباق والأقمار الصناعية

📐 المتغيرات الأساسية

φخط عرض الموقع الجغرافي (Latitude)
λsiteخط طول الموقع الجغرافي (Longitude)
λsatخط طول القمر الصناعي المداري
Δλفرق الطول = λsat - λsite
Rنصف قطر الأرض ≈ 6378 كم
hارتفاع القمر الصناعي ≈ 35786 كم (المدار الثابت)

المصدر: ITU-R S.1541 / Satellite Communications by Dennis Roddy

🧭 الزاوية الأفقية (Azimuth)

Az = 180° + arctan( tan(Δλ) / sin(φ) )

ملاحظة: للمواقع في نصف الكرة الشمالي (مثل سوريا) تستخدم هذه الصيغة. إذا كانت النتيجة سالبة، أضف 360°.

مثال تطبيقي (دمشق - نايل سات)
φ = 33.5°, λsite = 36.3°, λsat = 7° غرب = -7°
Δλ = -7° - 36.3° = -43.3°
Az = 180° + arctan( tan(-43.3°) / sin(33.5°) ) = 180° + arctan(-1.706) = 120.3°
النتيجة: بما أن القمر غرباً، النتيجة الفعلية هي 360° - 120.3° = 239.7° (جنوب غرب)

المصدر: Satellite Communications, Dennis Roddy, Chapter 4

⬆️ الزاوية العمودية (Elevation)

El = arctan( (cos(Δλ) × cos(φ) - 0.151) / √(1 - (cos(Δλ) × cos(φ))²) )
مثال تطبيقي (دمشق - نايل سات)
cos(Δλ) = 0.728, cos(φ) = 0.834 → 0.607
El = arctan((0.607 - 0.151) / √(1 - 0.368)) = arctan(0.456 / 0.795) = 29.9°

المصدر: Satellite Communications, Dennis Roddy, Chapter 4

🔄 إمالة الـ LNB (Skew)

Skew = arctan( sin(Δλ) / tan(φ) )
مثال تطبيقي (دمشق - نايل سات)
sin(43.3°) = 0.686, tan(33.5°) = 0.662 → Skew = arctan(1.036) = 46°

المصدر: Eutelsat Technical Documentation

📡 حساب المسافة إلى القمر

d = √( (R+h)² + R² - 2R(R+h) × cos(Δλ) × cos(φ) )
لنايل سات في دمشق: المسافة ≈ 37800 كم

المصدر: Satellite Communications, Dennis Roddy
